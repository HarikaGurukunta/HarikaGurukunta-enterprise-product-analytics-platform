# Resume Project Description

## Enterprise Product Analytics Platform

Developed an end-to-end analytics platform using Python, SQL, and Power BI to analyze customer behavior, product performance, conversion funnels, retention metrics, and revenue trends.

### Responsibilities

- Designed dimensional data models for customers, products, orders, and web events.
- Developed Python-based ETL pipelines for data cleansing and transformation.
- Generated synthetic datasets for customer and product analytics.
- Created SQL analytics solutions for CLV, funnel analysis, cohort analysis, and retention reporting.
- Designed Power BI dashboards for executive KPI monitoring and business insights.
- Implemented customer segmentation and product performance analytics.

### Technologies

- Python
- Pandas
- NumPy
- SQL
- Power BI
- Data Modeling
- ETL
- Analytics Engineering

### Business Value

Provided actionable insights into revenue growth, customer retention, product performance, and conversion optimization.
